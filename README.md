# EmoVision

Reconnaissance des émotions faciales en temps réel par CNN sur FER-2013.

Projet réalisé dans le cadre de l'UE Projet - Double Licence Mathématiques-Informatique L3, Aix-Marseille Université.

## Auteurs
- Bardak Ali Sait
- Bouslah Rayane

Encadrante : Line Jakubiec-Jamet

## Structure

- `train/` : entraînement du CNN
- `predict/` : module d'inférence
- `gui/` : interface graphique
- `tests/` : tests unitaires et d'intégration
- `docs/` : rapport et documentation
